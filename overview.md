# Overview

VeilBridge is the privacy layer for multichain DeFi.

### Core Features
- Fully anonymous cross-chain transfers
- Bidirectional: works from/to any supported chain
- Zero-knowledge anonymity pools anchored on Ethereum
- Fixed-denomination pools for maximum unlinkability
- Low fees via Layer 2 scaling (Arbitrum, Optimism, Base)

### Supported at Launch
**Assets:** ETH, WETH, USDC, USDT, DAI  
**Chains:** Ethereum, Polygon, BSC, Arbitrum, Optimism, Base

More chains coming post-launch (Solana, Avalanche, zkSync, Base, etc.).

### Mission
Deliver unbreakable financial privacy in an interconnected multichain world.
