# VeilBridge

<div align="center">
  <img src="assets/logo.png" alt="VeilBridge Logo" width="250"/>
</div>

**Cross chains. Erase traces.**

VeilBridge is a fully private, bidirectional multichain bridge powered by zero-knowledge cryptography.

Deposit from any supported chain → Anonymize on Ethereum → Withdraw to any other chain and address.  
No transaction links. No traces left behind.

**Mainnet Launch:** 2026
